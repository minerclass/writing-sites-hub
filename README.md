# Writing Sites Hub

Public-facing catalog for Micah Miner writing-adjacent GitHub Pages projects that are not part of the qualifying paper or dissertation companion-site ecosystem.

## Scope

This hub collects:

- Practitioner-facing writing and frameworks
- Conference and webinar sites
- District leadership briefs and public reports
- Portfolio and publication pathways
- Tools that support public writing or professional learning

Dissertation instruments, qualifying-paper companions, and research-methods pages belong in the Dissertation Sites Dashboard instead.

## Local Preview

Open `index.html` directly, or serve the workspace root locally and visit:

`http://127.0.0.1:8767/writing-sites-hub/index.html`

## Publishing

For GitHub Pages, this folder can become its own standalone repository with `index.html` at the repository root.
